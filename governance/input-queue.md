# Human Input Queue (Ask the PM)

Purpose: track the minimum human inputs required to unblock high-quality agent outputs.

Use this file as the default queue for "What do you need from me today?".

---

## How to use

- Agent adds/updates items whenever human input is required.
- PM answers inline (or via chat), then item is marked complete.
- Keep queue short and prioritised; archive completed items weekly.
