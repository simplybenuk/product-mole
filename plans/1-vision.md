# Product Vision

## Product

Mole is a portable, local-first context operating system for product managers and AI assistants.

It helps product teams capture messy inputs, distil them into structured context, and create stronger product outputs from the context already held in the repository.

## Core Loop

The product loop is:

1. Capture low-friction raw inputs.
2. Synthesise useful evidence and context.
3. Retrieve top-down from summaries and indexes.
4. Create output-ready artefacts such as roadmaps, specs, decision briefs, and strategy memos.

## Direction

The project should mature from a useful repository template into a practical open-source tool with:

- a durable file-native context architecture
- a clear workflow model for humans and agents
- a lightweight `mole` command surface
- safe lifecycle management for working instances

## Non-Goals

- Do not replace the file system as the source of truth.
- Do not require a hosted SaaS service to get value.
- Do not hide important product context inside opaque chat history.
- Do not overwrite local working-instance customisations without review.
